Bootstrap slideshow demo

This page demonstrates a modified slideshow with captions and a slideshow in a shadowbox.

Overrides Bootstrap 3 to move the captions and indicators under the images and removes the shading from the carousel navigators.

Hoping this will save somebody else some time.

All the action is in the stylesheet "css/s.css"

The images are Creative Commons license by Nic Lindh.
